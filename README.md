# Intro-Extro Club Official Landing Page

A responsive, mobile-first single-page website built for Intro-Extro Club to centralize information and improve member engagement

![Intro-ExtroScreenshot](https://github.com/user-attachments/assets/aa6d1f93-7eab-4e2e-9044-8b69b96765a3)

## About the Project
This project was designed using a **Mobile-First** methodology, ensuring optimal performance and user experience on smartphones before scaling up to tablets and desktops. The site is built entirely with **semantic HTML5** and **modern CSS3**, utilizing **Flexbox** and **Grid** for layout architecture without relying on JavaScript or heavy frameworks.

## Key Features
- **Mobile-First Architecture:** Base styles optimized for small screens, with progressive enhancement via media queries.
- **Modern Layouts:** 
  - **CSS Grid** used for the main content structure and galleries.
  - **Flexbox** used for navigation bars, cards, and component alignment.
- **Third-Party Integration:** Seamless embedding of external tools (Google Maps, Event Calendars) using `iframe`.
- **Performance:** Lightweight code with fast load times due to zero JavaScript dependencies.
- **Responsiveness:** Fully fluid design that adapts to any screen size.

## Technologies Used
- **HTML5** (Semantic Structure)
- **CSS3** (Styling, Variables, Animations)
- **CSS Grid** & **Flexbox** (Layout Systems)
- **Media Queries** (Breakpoints)
- **Git & GitHub** (Version Control & Hosting)
